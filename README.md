# My Portfolio

This repository is a modern, responsive personal portfolio template built with React + Vite + TailwindCSS and Framer Motion.

Features
- Animated Hero and micro-interactions using `framer-motion`
- Glassmorphism visual style with Tailwind utilities
- Projects grid, services, animated experience timeline, contact form with validation
- Cursor follower and subtle parallax/hover micro-interactions

See the `Setup & Run Instructions` section in this README for how to run locally and build for production.
